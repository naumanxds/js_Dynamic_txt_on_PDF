How To Run Project
===================

-Right Click on the index.html file
-Select Open with Google Chrome / Any Other Web Browser
-Enjoy 


How To Use
===========

-Select Any Option (health/education/cheque)
-Insert Text into the fields
-Click (Export to PDF)
